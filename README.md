# Course React Dapp

## Run locally

1. Clone the repo

```sh
git clone 
```

2. Install the dependencies

```sh
npm install

# or

yarn
```

3. Start the local test node

```sh
npx hardhat node
```

4. Compile the contract

```sh
npx hardhat compile
```
5. Test the contract

```sh
npx hardhat test
```

6. Deploy the contract

```sh
npx hardhat run scripts/deploy.js --network localhost
```

7. Run the app

```sh
npm start
```
